‚ùì Common Student Questions
"Why can't we just use Google Drive/Dropbox?"
Explain that while file sharing services are useful, they lack:
  • Detailed change tracking
  • Branch management for features
  • Conflict resolution tools
  • Code review capabilities
"Do we need to memorize all these commands?"
Reassure students that:
  • VS Code provides a GUI interface
  • They'll learn commands through regular use
  • Cheat sheets are available
  • Auto-completion helps with commands
üí° Teaching Strategies
  • Start with visual explanations
    • Draw diagrams on the board
    • Use the timeline visualization in VS Code
    • Show before/after scenarios
  • Use real-world analogies
    • Video game save points
    • Document revision history
    • Photo editing undo history
  • Interactive demonstrations
    • Live coding sessions
    • Group exercises
    • Pair programming activities
