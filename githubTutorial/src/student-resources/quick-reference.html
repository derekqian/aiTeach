Essential Git Commands

Repository Setup

git init Create a new Git repository
git clone URL Copy a repository from GitHub

Basic Operations

git status Check repository status
git add . Stage all changes
git commit -m "message" Save staged changes

Branching

git branch List branches
git branch name Create new branch
git checkout name Switch to branch

Remote Operations

git push Upload changes
git pull Download changes
git fetch Check for changes

VS Code Shortcuts

General

Ctrl + ` Toggle terminal
Ctrl + Shift + G Open Source Control
Ctrl + Shift + E Open Explorer

Source Control

Ctrl + Enter Commit staged changes
Alt + Arrow Navigate changes

Common Workflows

Starting a New Feature

  1. Create new branch: git checkout -b feature-name
  2. Make changes to files
  3. Stage changes: git add .
  4. Commit changes: git commit -m "Add feature"
  5. Push to GitHub: git push origin feature-name

Updating Your Code

  1. Get latest changes: git pull origin main
  2. Switch to your branch: git checkout your-branch
  3. Merge main: git merge main
  4. Resolve any conflicts
  5. Push updates: git push

Troubleshooting Tips

Undo Last Commit

git reset --soft HEAD~1

Discard Changes

git checkout -- filename

View Commit History

git log --oneline
