Homework Assignments

Practice your Git and GitHub skills with these hands-on assignments.

Assignment 1: Repository Setup and Basic Operations

Duration: 30 minutes Difficulty: Beginner

Learn the basics of creating and managing a Git repository.

Requirements:

  • Create a new repository on GitHub
  • Clone it to your local machine
  • Create a README.md file with your name and course info
  • Add and commit the file
  • Push changes to GitHub

Submission Guidelines:

Submit the URL of your GitHub repository.

Assignment 2: Branching and Merging

Duration: 45 minutes Difficulty: Intermediate

Practice creating branches and merging changes.

Requirements:

  • Create a new feature branch
  • Add a new HTML file with a simple webpage
  • Create a CSS file and style your webpage
  • Commit changes to your feature branch
  • Create a pull request to merge into main
  • Add your teacher as a reviewer

Submission Guidelines:

Submit the URL of your pull request.

Assignment 3: Collaboration Exercise

Duration: 60 minutes Difficulty: Advanced

Work with a partner to simulate real-world collaboration.

Requirements:

  • Fork your partner's repository
  • Create a new feature branch
  • Add a new feature to their project
  • Submit a pull request to their repository
  • Review and comment on your partner's pull request
  • Resolve any merge conflicts

Submission Guidelines:

Submit both the URL of your pull request and your review of your partner's pull request.

Assignment 4: Project Portfolio

Duration: 90 minutes Difficulty: Advanced

Create a personal portfolio using GitHub Pages.

Requirements:

  • Create a new repository for your portfolio
  • Set up GitHub Pages
  • Create a multi-page website showcasing your projects
  • Include proper Git commit history
  • Add project documentation in markdown
  • Implement responsive design

Submission Guidelines:

Submit the URL of your GitHub Pages site and repository.
