Documentation Site Template

Use Template

A template for creating project documentation using GitHub Pages and Jekyll.

Includes:

  • Jekyll configuration
  • Documentation structure
  • Search functionality
  • Responsive theme
  • Navigation system

Config Example

title: Project Documentation
description: Project documentation template
theme: jekyll-theme-minimal

defaults:
  - scope:
      path: ""
    values:
      layout: default

plugins:
  - jekyll-search
  - jekyll-sitemap
