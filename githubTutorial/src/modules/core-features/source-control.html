Source Control Tab in VS Code

Accessing Source Control

Opening Source Control

  • Click the Source Control icon in the Activity Bar (looks like a branch)
  • Use keyboard shortcut: Ctrl+Shift+G (Cmd+Shift+G on macOS)

Understanding the Interface

Main Sections

Changes
Shows files you've modified but haven't committed yet
Staged Changes
Files you've selected to include in your next commit
Merge Changes
Appears when resolving conflicts during merges

File Status Icons

Icon Meaning
U Untracked - New file
M Modified - Changed file
D Deleted file
A Added - Staged new file

Common Actions

Working with Changes

  • Stage Changes:
    • Click the + (plus) icon next to a file
    • Click the + icon in the Changes header to stage all
  • Unstage Changes:
    • Click the - (minus) icon next to a staged file
    • Click the - icon in the Staged Changes header to unstage all
  • Committing:
    • Type your commit message in the input box
    • Press Ctrl+Enter (Cmd+Enter on macOS) to commit
    • Click the checkmark icon to commit

Viewing Changes

Diff View Features

  • View Changes: Click any modified file to see changes
  • Side-by-side Diff: Click the Split Editor icon
  • Line Changes:
    • Red: Removed lines
    • Green: Added lines
    • Blue: Modified lines
  • Stage Partial Changes: Click the + icon next to specific lines
