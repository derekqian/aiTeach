Integrated Terminal Essentials

Opening the Terminal

There are several ways to open the integrated terminal in VS Code:

  • Press Ctrl + ` (backtick) or Cmd + ` on macOS
  • Use the menu: View ‚Üí Terminal
  • Use the command palette (Ctrl+Shift+P or Cmd+Shift+P) and type "Terminal: Create New Terminal"

Terminal Features in VS Code

Multiple Terminals

  • Create new terminal: Ctrl+Shift+`
  • Switch between terminals: Ctrl+`
  • Split terminal: Click the split icon or use Ctrl+Shift+5

Command History

  • Up/Down arrows: Browse previous commands
  • Ctrl+R: Search command history
  • Type part of a command and press Up to search related commands

Tab Completion

  • Press Tab to complete file/folder names
  • Double Tab to show all possible completions
  • Works with Git commands and branch names
