Essential Git Commands

Basic Git Commands

Repository Setup

Command Description Example
git init Create a new Git repository git init
git clone Copy a repository from GitHub git clone https://github.com/user/repo.git

Basic Operations

git status Check repository status git status
git add Stage changes for commit git add file.txt or git add .
git commit Save staged changes git commit -m "Add new feature"

Working with Branches

Command Description Example
git branch List, create, or delete branches git branch feature-login
git checkout Switch branches or restore files git checkout feature-login
git merge Combine branches git merge feature-login

Remote Operations

Command Description Example
git remote Manage remote repositories git remote -v
git push Upload changes to remote git push origin main
git pull Download changes from remote git pull origin main
git fetch Get remote changes without merging git fetch origin

Viewing History

Command Description Example
git log View commit history git log or git log --oneline
git diff Show changes between commits git diff HEAD~1
git show Show commit details git show abc123

Command Tips

Helpful Tips

  • Use git help command to get help about any command
  • Press Q to exit from log or help pages
  • Use Tab completion to avoid typing full commands
  • Add -h to any command for quick help
