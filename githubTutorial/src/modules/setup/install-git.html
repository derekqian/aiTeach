Installing Git

Download and Install Git

Windows Installation

  1. Visit git-scm.com
  2. Download the latest Git for Windows installer
  3. Run the installer with these recommended options:
    • Use Git from Git Bash and Windows Command Prompt
    • Use the OpenSSL library
    • Checkout Windows-style, commit Unix-style line endings
    • Use MinTTY terminal
    • Enable file system caching

macOS Installation

Option 1: Homebrew (Recommended)

# Install Homebrew first (if not installed)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install Git
brew install git

Option 2: Command Line Tools

Open Terminal and run:

xcode-select --install

Linux Installation

Ubuntu/Debian:

sudo apt update
sudo apt install git

Fedora:

sudo dnf install git

Verifying the Installation

Open your terminal (Command Prompt, Terminal, or Git Bash) and run:

git --version

You should see something like: git version 2.40.0

VS Code Integration

After installing Git:

  1. Open VS Code
  2. The Source Control icon should appear in the sidebar
  3. Click it to start using Git in VS Code

If you don't see the Source Control icon, try:

  • Restarting VS Code
  • Verifying Git is in your system's PATH
