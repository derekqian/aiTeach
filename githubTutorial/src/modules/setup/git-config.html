Configuring Git

Initial Setup

Essential Configuration

Open your terminal in VS Code (Press Ctrl+` or Cmd+`) and set up your identity:

# Set your name
git config --global user.name "Your Name"

# Set your email
git config --global user.email "your.email@example.com"

üí° Use the same email address you'll use for GitHub or other Git services.

Recommended Settings

Default Branch Name

git config --global init.defaultBranch main

Default Editor

git config --global core.editor "code --wait"

Line Endings

git config --global core.autocrlf true
git config --global core.autocrlf input

Verifying Your Configuration

Check your settings:

# View all settings
git config --list

# View specific setting
git config user.name
git config user.email

VS Code Git Settings

  1. Open VS Code Settings (Ctrl+, or Cmd+,)
  2. Search for "git"
  3. Recommended settings to enable:
    • Git: Enable Smart Commit
    • Git: Confirm Sync
    • Git: Autofetch
