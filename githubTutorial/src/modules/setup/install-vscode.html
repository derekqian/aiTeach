Installing Visual Studio Code

Getting Started

Download VS Code

  1. Visit the VS Code website
  2. Click the download button for your operating system:
    • Windows: .exe installer
    • macOS: .dmg file
    • Linux: .deb/.rpm package or snap

Installation Steps

Windows Installation

  1. Double-click the downloaded .exe file
  2. Accept the license agreement
  3. Choose the installation location (default is recommended)
  4. Select Start Menu folder
  5. Check the additional tasks you want:
    • "Open with Code" action
    • Add to PATH
  6. Click Install

macOS Installation

  1. Open the downloaded .dmg file
  2. Drag VS Code to the Applications folder
  3. Optional: Add VS Code to Dock

Linux Installation

Ubuntu/Debian:

sudo apt update
sudo apt install code

Snap:

sudo snap install code --classic

Verifying Installation

  1. Open VS Code
  2. Press Ctrl+` (or Cmd+` on macOS) to open the integrated terminal
  3. Type code --version to verify the installation
